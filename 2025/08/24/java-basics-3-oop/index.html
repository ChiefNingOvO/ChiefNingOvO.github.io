一、类与对象

类是对象的模板，对象是类的实例。

二、封装

用 private 保护数据，通过 public 方法对外暴露有限访问，这就是封装。

修饰符 当前类 同包 子类 其他包
private ✅
默认（无修饰符） ✅ ✅
protected ✅ ✅ ✅
public ✅ ✅ ✅ ✅

三、继承

用 extends 实现继承，子类获得父类的所有属性和方法（构造方法除外）。

super 关键字

Java 只支持单继承

一个类只能有一个直接父类，但可以通过接口实现多继承的效果。

四、多态

多态的核心是：父类引用指向子类对象，调用方法时执行子类的重写版本。

多态的三个条件：

  1. 继承或实现
  2. 重写父类方法
  3. 父类引用指向子类对象

五、抽象类

用 abstract 声明，不能实例化，可以含抽象方法（没有方法体）。

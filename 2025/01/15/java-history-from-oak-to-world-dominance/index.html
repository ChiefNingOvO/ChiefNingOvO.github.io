еј•иЁҖ

Java иҮӘ 1995 е№ҙй—®дё–д»ҘжқҘпјҢе·Із»Ҹиө°иҝҮдәҶиҝ‘дёүеҚҒе№ҙзҡ„еҸ‘еұ•еҺҶзЁӢгҖӮд»ҺжңҖеҲқдёәе®¶з”Ёз”өеҷЁи®ҫи®Ўзҡ„еөҢе…ҘејҸиҜӯиЁҖпјҢеҲ°еҰӮд»Ҡж”Ҝж’‘е…Ёзҗғж•°зҷҫдёҮдјҒдёҡеә”з”Ёзҡ„ж ёеҝғжҠҖжңҜпјҢJava зҡ„жј”иҝӣеҸІжң¬иә«е°ұжҳҜдёҖйғЁиҪҜд»¶е·ҘзЁӢзҡ„ж•ҷз§‘д№ҰгҖӮжң¬ж–Үе°ҶеӣһйЎҫ Java д»Һ Oak йЎ№зӣ®иө·жӯҘзҡ„е…ій”®еҺҶеҸІиҠӮзӮ№пјҢжўізҗҶдё»иҰҒзүҲжң¬йҮҢзЁӢзў‘пјҢе№¶еҲҶжһҗ Java дёәдҪ•иҮід»Ҡдҫқз„¶зЁіеұ…зј–зЁӢиҜӯиЁҖжҺ’иЎҢжҰңзҡ„еүҚеҲ—гҖӮ

Oak йЎ№зӣ®зҡ„иө·жәҗпјҲ1991-1994пјү

ж—¶й—ҙеӣһеҲ° 1991 е№ҙпјҢSun Microsystems зҡ„е·ҘзЁӢеёҲ James Gosling еёҰйўҶдёҖдёӘеҗҚдёәвҖқGreen TeamвҖқзҡ„е°Ҹз»„пјҢиҜ•еӣҫдёәж¶Ҳиҙ№з”өеӯҗдә§е“ҒпјҲеҰӮжңәйЎ¶зӣ’гҖҒPDAпјүејҖеҸ‘дёҖз§ҚеҸҜз§»жӨҚзҡ„зј–зЁӢиҜӯиЁҖгҖӮд»–е°Ҷиҝҷй—ЁиҜӯиЁҖе‘ҪеҗҚдёә OakвҖ”вҖ”зҒөж„ҹжқҘиҮӘд»–еҠһе…¬е®ӨзӘ—еӨ–зҡ„дёҖжЈөж©Ўж ‘гҖӮ

з„¶иҖҢ Oak иҝҷдёӘеҗҚеӯ—еҫҲеҝ«ж’һдёҠдәҶе•Ҷж Үзә зә·пјҢеӣўйҳҹеңЁдёҖж¬Ўе’–е•ЎйҰҶи®Ёи®әеҗҺе°Ҷе…¶ж”№еҗҚдёә JavaпјҲеҚ°еәҰе°јиҘҝдәҡзҲӘе“ҮеІӣзҡ„е’–е•ЎпјүгҖӮиҝҷдёӘе‘ҪеҗҚд№ҹдёәеҗҺжқҘ Java ж Үеҝ—жҖ§зҡ„е’–е•ЎжқҜ LOGO еҹӢдёӢдәҶдјҸз¬”гҖӮ

Oak/Java зҡ„ж ёеҝғзҗҶеҝөеңЁеҪ“ж—¶жһҒе…·еүҚзһ»жҖ§пјҡвҖңWrite Once, Run AnywhereвҖқпјҲдёҖж¬Ўзј–еҶҷпјҢеҲ°еӨ„иҝҗиЎҢпјүгҖӮеӣўйҳҹи®ҫи®ЎдәҶдёҖеҘ—еӯ—иҠӮз ҒпјҲbytecodeпјүжҢҮд»ӨйӣҶпјҢй…ҚеҗҲдёҖдёӘиҷҡжӢҹжү§иЎҢзҺҜеўғвҖ”вҖ”еҚіеҗҺжқҘзҡ„ Java иҷҡжӢҹжңәпјҲJVMпјүгҖӮжәҗзЁӢеәҸиў«зј–иҜ‘дёәдёҺе№іеҸ°ж— е…ізҡ„еӯ—иҠӮз ҒпјҢз”ұеҗ„е№іеҸ°зҡ„ JVM иҙҹиҙЈи§ЈйҮҠжү§иЎҢгҖӮиҝҷдёҖи®ҫи®ЎдҪҝ Java еңЁи·Ёе№іеҸ°ж–№йқўе…·еӨҮдәҶеӨ©з„¶дјҳеҠҝгҖӮ

Java 1.0 зҡ„еҸ‘еёғдёҺж—©жңҹзҲҶеҸ‘пјҲ1995-1998пјү

1995 е№ҙ 5 жңҲ 23 ж—ҘпјҢSun еңЁ SunWorld еӨ§дјҡдёҠжӯЈејҸеҸ‘еёғ Java 1.0гҖӮзҙ§йҡҸе…¶еҗҺпјҢNetscape Navigator жөҸи§ҲеҷЁе®ЈеёғеҶ…зҪ® Java ж”ҜжҢҒпјҢиҝҷи®© Java д»ҘвҖқAppletвҖқзҡ„еҪўејҸиҝ…йҖҹеҚ жҚ®дәҶ Web ејҖеҸ‘иҖ…зҡ„и§ҶйҮҺгҖӮ1996 е№ҙпјҢJDK 1.0.2 еҸ‘еёғпјҢJava жӯЈејҸиө°еҮәе®һйӘҢе®ӨгҖӮ

Java 1.1пјҲ1997пјүжҳҜдёҖдёӘйҮҚиҰҒзҡ„еўһйҮҸзүҲжң¬пјҢеј•е…ҘдәҶпјҡ

  • JDBCпјҡJava ж•°жҚ®еә“иҝһжҺҘ API
  • RMIпјҡиҝңзЁӢж–№жі•и°ғз”Ё
  • еҶ…йғЁзұ»пјҲInner Classпјү
  • еҸҚе°„пјҲReflectionпјү
  • JavaBeans з»„д»¶жЁЎеһӢ

иҝҷдәӣзү№жҖ§дҪҝ Java д»ҺвҖқжөҸи§ҲеҷЁйҮҢзҡ„еҠЁз”»и„ҡжң¬вҖқзңҹжӯЈиң•еҸҳдёәдёҖй—ЁйҖӮз”ЁдәҺдјҒдёҡзә§ејҖеҸ‘зҡ„йҖҡз”ЁиҜӯиЁҖгҖӮ

Java 2 е№іеҸ°зҡ„иҜһз”ҹпјҲ1998-2004пјү

1998 е№ҙ 12 жңҲпјҢJDK 1.2 еҸ‘еёғпјҢSun е°Ҷе…¶йҮҚж–°е‘ҪеҗҚдёә Java 2гҖӮиҝҷж¬ЎеҸ‘еёғжңҖе…·ж·ұиҝңж„Ҹд№үзҡ„дёҚжҳҜиҜӯиЁҖзү№жҖ§жң¬иә«пјҢиҖҢжҳҜе№іеҸ°еҲ’еҲҶзҡ„дҪ“зі»еҢ–пјҡ

зүҲжң¬ е®ҡдҪҚ з”ЁйҖ”
J2SE Standard Edition жЎҢйқўдёҺж ҮеҮҶеә”з”Ё
J2EE Enterprise Edition дјҒдёҡзә§жңҚеҠЎеҷЁз«Ҝеә”з”Ё
J2ME Micro Edition з§»еҠЁдёҺеөҢе…ҘејҸи®ҫеӨҮ

JDK 1.2 еңЁжҠҖжңҜеұӮйқўеҗҢж ·дә®зңјпјҡSwing еӣҫеҪўеә“гҖҒCollections FrameworkгҖҒд»ҘеҸҠ strictfp е…ій”®еӯ—йғҪеңЁиҝҷдёӘзүҲжң¬иў«еј•е…ҘгҖӮеҗҺз»ӯзҡ„ JDK 1.3пјҲ2000пјүдёҺ 1.4пјҲ2002пјүз»§з»ӯдё°еҜҢз”ҹжҖҒпјҢ1.4 еёҰжқҘдәҶ assert е…ій”®еӯ—гҖҒNIOгҖҒд»ҘеҸҠжӯЈеҲҷиЎЁиҫҫејҸж”ҜжҢҒгҖӮ

Java 5пјҡиҜӯиЁҖзҺ°д»ЈеҢ–зҡ„иҪ¬жҠҳзӮ№пјҲ2004пјү

2004 е№ҙеҸ‘еёғзҡ„ J2SE 5.0пјҲзүҲжң¬еҸ·д»Һ 1.4 зӣҙжҺҘи·іеҲ° 5.0пјүиў«и§Ҷдёә Java иҜӯиЁҖзҺ°д»ЈеҢ–зҡ„дёҖж¬ЎйҮҚеӨ§йЈһи·ғгҖӮе®ғеј•е…ҘдәҶдёҖж•ҙеҘ—жәҗиҮӘеҮҪж•°ејҸе’ҢжіӣеһӢзј–зЁӢзҡ„жҖқжғіпјҡ

жіӣеһӢеҪ»еә•ж”№еҸҳдәҶ Java йӣҶеҗҲзҡ„дҪҝз”Ёж–№ејҸвҖ”вҖ”д»ҺжӯӨе‘ҠеҲ«дәҶејәеҲ¶зұ»еһӢиҪ¬жҚўе’ҢиҝҗиЎҢж—¶ ClassCastExceptionгҖӮжіЁи§ЈеҲҷдёәеҗҺжқҘзҡ„ SpringгҖҒHibernate зӯүжЎҶжһ¶жҸҗдҫӣдәҶеЈ°жҳҺејҸзј–зЁӢзҡ„еҹәзҹігҖӮ

Java 8пјҡеҮҪж•°ејҸйқ©е‘ҪпјҲ2014пјү

еҰӮжһңиҜҙ Java 5 ж”№еҸҳдәҶ Java зҡ„зұ»еһӢзі»з»ҹпјҢйӮЈд№Ҳ Java 8 еҲҷж”№еҸҳдәҶ Java зҡ„зј–зЁӢиҢғејҸгҖӮ2014 е№ҙеҸ‘еёғзҡ„ Java 8 еј•е…ҘдәҶдёүдёӘйҮҚзЈ…зү№жҖ§пјҡ

Lambda иЎЁиҫҫејҸдёҺеҮҪж•°ејҸжҺҘеҸЈпјҡ

Stream APIпјҡ жҸҗдҫӣдәҶдёҖз§ҚеЈ°жҳҺејҸзҡ„гҖҒж”ҜжҢҒе№¶иЎҢеӨ„зҗҶзҡ„ж•°жҚ®ж“ҚдҪңж–№ејҸпјҢе°ҶеҮҪж•°ејҸзј–зЁӢжҺЁеҗ‘дәҶ Java ејҖеҸ‘иҖ…зҡ„ж—Ҙеёёе·ҘдҪңжөҒгҖӮ

ж–°зҡ„ж—Ҙжңҹж—¶й—ҙ APIпјҲjava.timeпјүпјҡ жәҗиҮӘ Joda-Time зҡ„и®ҫи®ЎпјҢеҪ»еә•и§ЈеҶідәҶ java.util.Date е’Ң Calendar зҡ„зәҝзЁӢе®үе…ЁдёҺжҳ“з”ЁжҖ§й—®йўҳгҖӮ

Java 8 жҳҜ Java еҺҶеҸІдёҠеҪұе“ҚжңҖж·ұиҝңзҡ„зүҲжң¬д№ӢдёҖпјҢзӣҙеҲ°д»ҠеӨ©д»ҚжңүеӨ§йҮҸдјҒдёҡйЎ№зӣ®иҝҗиЎҢеңЁ Java 8 дёҠгҖӮ

д»Һ Java 9 еҲ° Java 17пјҡжЁЎеқ—еҢ–дёҺ LTS иҠӮеҘҸ

Java 9пјҲ2017пјү зҡ„жңҖеӨ§дә®зӮ№жҳҜ Project Jigsaw жЁЎеқ—еҢ–зі»з»ҹгҖӮmodule-info.java дҪҝејҖеҸ‘иҖ…иғҪеӨҹжҳҫејҸеЈ°жҳҺжЁЎеқ—й—ҙзҡ„дҫқиө–е…ізі»пјҢи§ЈеҶідәҶ JAR ең°зӢұй—®йўҳгҖӮ

Oracle еҗҢж—¶е®ЈеёғдәҶж–°зҡ„еҸ‘еёғиҠӮеҘҸпјҡжҜҸ 6 дёӘжңҲеҸ‘еёғдёҖдёӘж–°зүҲжң¬пјҢжҜҸ 3 е№ҙеҸ‘еёғдёҖдёӘ LTSпјҲй•ҝжңҹж”ҜжҢҒпјүзүҲжң¬гҖӮ

Java 11пјҲ2018пјҢLTSпјү жҳҜз¬¬дёҖдёӘ LTS зүҲжң¬пјҢеҢ…еҗ« HTTP Client APIгҖҒvar еұҖйғЁеҸҳйҮҸзұ»еһӢжҺЁж–ӯпјҲvar list = new ArrayList<String>();пјүпјҢд»ҘеҸҠ Flight Recorder зҡ„ејҖжәҗеҢ–гҖӮ

Java 17пјҲ2021пјҢLTSпјү еҲҷиў«и§Ҷдёә Java 8 д№ӢеҗҺзҡ„дёӢдёҖд»Јдё»еҠӣзүҲжң¬пјҢеёҰжқҘдәҶпјҡеҜҶе°Ғзұ»пјҲSealed ClassesпјүгҖҒPattern Matching for instanceofгҖҒRecordsпјҲи®°еҪ•зұ»пјүгҖҒд»ҘеҸҠ Switch иЎЁиҫҫејҸзӯүжҸҗеҚҮејҖеҸ‘иҖ…е№ёзҰҸж„ҹзҡ„е…ій”®зү№жҖ§гҖӮ

Java 21 дёҺиҷҡжӢҹзәҝзЁӢпјҲ2023пјү

Java 21пјҲ2023пјҢLTSпјү дәӨд»ҳдәҶ Project Loom зҡ„ж ёеҝғжҲҗжһңвҖ”вҖ”иҷҡжӢҹзәҝзЁӢпјҲVirtual ThreadsпјүгҖӮиҷҡжӢҹзәҝзЁӢе°Ҷ JVM дёӯзҡ„зәҝзЁӢжҠҪиұЎдёҺж“ҚдҪңзі»з»ҹзәҝзЁӢи§ЈиҖҰпјҢе…Ғи®ёж•°д»ҘзҷҫдёҮи®Ўзҡ„иҪ»йҮҸзә§зәҝзЁӢеҗҢж—¶иҝҗиЎҢпјҢд»Һж №жң¬дёҠж”№еҸҳдәҶ Java жңҚеҠЎеҷЁз«Ҝй«ҳе№¶еҸ‘зј–зЁӢзҡ„жЁЎејҸгҖӮжӯӨеӨ–пјҢJava 21 иҝҳеёҰжқҘдәҶ Record PatternsгҖҒPattern Matching for switch зӯүзү№жҖ§пјҢдҪҝ Java еңЁиҜӯиЁҖиЎЁиҫҫеҠӣдёҠиҝӣдёҖжӯҘйҖјиҝ‘ Kotlin е’Ң ScalaгҖӮ

Oracle ж”¶иҙӯдёҺ OpenJDK ж—¶д»Ј

2009-2010 е№ҙпјҢOracle ж”¶иҙӯдәҶ Sun MicrosystemsпјҢJava зҡ„жүҖжңүжқғйҡҸд№ӢиҪ¬з§»гҖӮиҝҷдёҖж”¶иҙӯдёҖеәҰеј•еҸ‘дәҶзӨҫеҢәзҡ„жӢ…еҝ§пјҢдҪҶ Oracle еңЁ 2017 е№ҙе°Ҷ Java EE з§»дәӨз»ҷдәҶ Eclipse еҹәйҮ‘дјҡпјҲеҚіеҗҺжқҘзҡ„ Jakarta EEпјүпјҢиҜҒжҳҺдәҶе…¶е№¶йқһе…ЁзӣҳжҺ§еҲ¶гҖӮ

OpenJDK жҳҜ Java ејҖжәҗеҸӮиҖғе®һзҺ°зҡ„ж ёеҝғгҖӮиҮӘ Java 7 иө·пјҢOpenJDK жҲҗдёәе®ҳж–№еҸӮиҖғе®һзҺ°гҖӮеҰӮд»ҠпјҢйҷӨ Oracle JDK еӨ–пјҢAmazon CorrettoгҖҒAzul ZuluгҖҒAdoptiumпјҲEclipse Temurinпјүзӯүдј—еӨҡеҸ‘иЎҢзүҲзҷҫиҠұйҪҗж”ҫпјҢJava з”ҹжҖҒжҜ”д»ҘеҫҖд»»дҪ•ж—¶еҖҷйғҪжӣҙеҠ ејҖж”ҫе’Ңз№ҒиҚЈгҖӮ

дёәд»Җд№Ҳ Java иҮід»Ҡдҫқз„¶ејәеӨ§

  1. JVM з”ҹжҖҒзҡ„дёҚеҸҜжӣҝд»ЈжҖ§пјҡKotlinгҖҒScalaгҖҒGroovyгҖҒClojure зӯү JVM иҜӯиЁҖе…ұдә«еҗҢдёҖдёӘз”ҹжҖҒеҹәеә§пјҢзӣёдә’еўһејәиҖҢйқһжӣҝд»ЈгҖӮ
  2. дјҒдёҡзә§дҝЎд»»пјҡй“¶иЎҢгҖҒдҝқйҷ©гҖҒз”өе•Ҷзӯүж ёеҝғзі»з»ҹзҡ„жҠҖжңҜйҖүеһӢд»ҘзЁіе®ҡжҖ§дёәжңҖй«ҳдјҳе…ҲпјҢJava зҡ„еҗ‘еҗҺе…је®№жҖ§жҸҗдҫӣдәҶй•ҝиҫҫж•°еҚҒе№ҙзҡ„жҠ•иө„дҝқжҠӨгҖӮ
  3. жҢҒз»ӯжј”иҝӣиғҪеҠӣпјҡд»ҺжіӣеһӢеҲ° LambdaпјҢд»ҺжЁЎеқ—еҢ–еҲ°иҷҡжӢҹзәҝзЁӢпјҢJava еңЁдҝқжҢҒе…је®№зҡ„еүҚжҸҗдёӢд»ҺжңӘеҒңжӯўзҺ°д»ЈеҢ–гҖӮ

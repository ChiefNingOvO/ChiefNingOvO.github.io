Claude Code 命令大全：分类速查与使用指南

发表于2026-05-08|更新于2026-05-13|AI
|总字数:2k|阅读时长:6分钟|浏览量:

Claude Code 提供了几十个内置命令，覆盖从会话管理到代码审查、从项目初始化到远程控制的方方面面。输入 / 即可查看所有可用命令。

本文将所有命令按用途分组，方便快速查找。

会话管理

命令 说明
/clear [name] 清空对话历史，开始新会话。之前的会话可通过 /resume 恢复。传参可为之前的对话命名
/resume [session] 按 ID 或名称恢复历史对话。不传参时打开交互选择器
/branch [name] 在当前对话处创建分支，切换到新分支的同时保留原对话，可用 /resume 返回
/rename [name] 重命名当前会话。不传参时从对话历史自动生成名称
/recap 生成当前会话的简短摘要。用于快速回顾
/export [filename] 导出当前对话为纯文本。指定文件名直接写入，否则弹出保存对话框
/compact [instructions] 将对话压缩总结，释放上下文窗口空间。可传参指定聚焦内容
/exit 退出 Claude Code

任务执行与工作流

命令 说明
/plan [description] 进入 Plan Mode。传参可直接开始规划指定任务
/goal [condition|clear] 设定目标，Claude 会持续工作直到条件满足。传 clear 可提前结束
/background [prompt] 将当前会话转为后台 agent 运行，释放终端。可用 claude agents 监控
/batch <instruction> 在代码库中并行执行大规模变更。自动分解任务，在每个独立 worktree 中运行子 agent
/loop [interval] [prompt] 按间隔重复执行指定提示。不传间隔时自动调整节奏
/agents 管理子 agent 配置
/tasks 列出和管理后台任务
/stop 停止当前后台会话。日志和 worktree 保留

代码操作与审查

命令 说明
/diff 交互式差异查看器。显示未提交的更改和每轮差异
/review [PR] 在本地审查 pull request
/security-review 分析待处理更改的安全漏洞。审查 git diff 识别注入、鉴权问题等
/simplify [focus] 审查最近修改的文件，检查代码重用、质量和效率问题并自动修复
/copy [N] 复制最后一条助手响应。传数字 N 复制第 N 条。有代码块时弹出交互选择器
/rewind 将代码或对话回退到上一个检查点。支持选择特定消息
/ultrareview [PR] 在云端沙箱中运行深度多 agent 代码审查
/ultraplan <prompt> 起草大规模变更计划，在浏览器中审查，然后远程执行或拉回终端

模型与推理配置

命令 说明
/model [model] 切换 AI 模型。对支持情况，可同时调整工作量级别
/effort [level|auto] 设置推理工作量级别：low / medium / high / xhigh / max。不传参时打开滑块
/fast [on|off] 切换快速模式
/config 打开设置界面，调整主题、模型、输出样式等偏好
/theme 切换颜色主题。支持 auto、浅色/深色、ANSI 和自定义主题
/color [color|default] 为当前会话设置提示栏颜色。可传 red / blue / green 等，不传参随机分配
/tui [default|fullscreen] 切换终端 UI 渲染模式。fullscreen 启用无闪烁全屏模式

项目初始化与内存

命令 说明
/init 初始化项目，创建 CLAUDE.md 指南文件
/memory 编辑 CLAUDE.md 和自动记忆条目
/mcp 管理 MCP server 连接和 OAuth 认证
/permissions 管理工具权限规则：允许、询问、拒绝。可查看和删除规则
/hooks 查看和管理工具事件 hook 配置
/plugin 管理 Claude Code 插件
/skills 查看已安装的 skills，按空格隐藏不想要的
/reload-plugins 重新加载所有插件，应用变更无需重启

信息与诊断

命令 说明
/context [all] 可视化当前上下文使用情况，显示各工具占用的 token 分布和优化建议
/usage 显示会话成本、计划用量限制和活跃度统计。支持 /cost 和 /stats 别名
/doctor 诊断并验证 Claude Code 安装和配置。按 f 自动修复报告的问题
/debug [description] 启用调试日志并分析会话日志排查问题。可描述具体问题来聚焦分析
/status 打开状态选项卡，显示版本、模型、账户和连接状态
/insights 分析 Claude Code 使用会话，生成项目领域、交互模式和摩擦点报告
/heapdump 输出 JavaScript 堆快照到桌面，用于诊断高内存占用

远程控制与跨设备

命令 说明
/teleport 将网络版 Claude Code 会话拉入当前终端。会同步分支和对话
/remote-control 使当前会话可从 claude.ai/code 远程控制
/desktop 在 Claude Code 桌面应用中继续当前会话（macOS / Windows）
/mobile 显示二维码下载 Claude 移动应用
/chrome 配置 Claude in Chrome 设置
/ide 管理 IDE 集成并显示状态

账户与计划

命令 说明
/login 登录 Anthropic 账户
/logout 退出登录
/upgrade 升级到更高计划层级
/privacy-settings 查看和更新隐私设置（Pro / Max 用户）
/extra-usage 配置额外用量，在达到速率限制后继续工作

GitHub 与 CI/CD

命令 说明
/autofix-pr [prompt] 为当前分支的 PR 启动自动修复。CI 失败或审查者评论时会推送修复
/install-github-app 为仓库安装 Claude GitHub Actions 应用
/install-slack-app 安装 Claude Slack 应用
/web-setup 将 GitHub 账户关联到网络版 Claude Code

交互与快捷操作

命令 说明
/btw <question> 提出旁路问题，不计入对话历史。适合临时性的小疑问
/help 查看帮助和命令列表
/statusline 自定义底部状态行。描述想显示的内容，Claude 自动生成脚本
/keybindings 打开或创建快捷键配置文件
/terminal-setup 配置终端快捷键（如 Shift+Enter）
/powerup 通过动画演示交互式发现 Claude Code 功能
/voice [hold|tap|off] 切换语音听写
/scroll-speed 调整鼠标滚轮滚动速度（仅全屏模式）
/focus 切换聚焦视图，只显示最近的提示和最终响应（仅全屏模式）

MCP 与 Skills

命令 说明
/fewer-permission-prompts 扫描历史记录，自动生成权限允许列表以减少弹窗
/claude-api [migrate|managed-agents-onboard] 加载 Claude API 参考资料。migrate 升级 API 代码到新模型
/team-onboarding 根据使用记录生成团队入职指南，含共享链接
/schedule [description] 创建在云端执行的定时任务（routines）
/mcp__<server>__<prompt> 调用已连接 MCP server 暴露的 prompt
